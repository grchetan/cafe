# ☕ Paaji Ka Bhaaji – Café Website Project

This project is a **frontend-based café website** developed as part of my web development learning journey. The goal was to practice modern design principles while building a creative, visually engaging, and user-friendly interface.

## 📌 Project Objective

To design and develop a fully responsive café website that showcases:
- Engaging homepage with strong branding
- Customer testimonials with creative tone
- Live Google Map integration for real-world feel
- Clean layout using Tailwind CSS utility classes
- Professional and quirky blend of content and UI


## 🚀 Tech Stack

- HTML5  
- Tailwind CSS  
- Google Maps Embed API  
- Git & GitHub

## 💡 Use Case

This website can be used as a concept for local cafés or coffee shop startups looking to establish a playful-yet-professional online presence.
